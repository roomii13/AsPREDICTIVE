# Propuesta comercial base

## Alcance

- Implementación de tablero de seguridad operacional
- Registro y priorización de incidentes
- Alertas predictivas
- Modelo de IA entrenado con datos reales
- Auditoría de acciones críticas
- Capacitación inicial

## Tiempos sugeridos

- Semana 1: relevamiento y parametrización
- Semana 2: carga inicial de datos y validación operativa
- Semana 3: entrenamiento ajustado y demo funcional
- Semana 4: salida a piloto

## Soporte

- soporte correctivo
- monitoreo básico
- respaldo y restauración
- ajustes de modelo programados

## Exclusiones sugeridas

- integración con sistemas legacy no relevados
- correo transaccional productivo
- infraestructura on-premise compleja
- cumplimiento regulatorio formal certificado por terceros

## Modalidad sugerida

- piloto pago de 60 a 90 días
- opción de continuidad mensual o implementación ampliada
